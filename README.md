Prerequisites

1.	Node Js
2.	Ganache
3.	MySQL
4.	Metamask - Browser plugin

Installation
1.	Open terminal in Client and Server folder.
a.	npm install
2.	In Client
a.	truffle migrate > only first time
b.	npm run dev
3.	 In Server
a.	node .
